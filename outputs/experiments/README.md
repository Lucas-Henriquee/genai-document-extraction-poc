# Saídas dos experimentos

Arquivos gerados pela POC durante os experimentos de latência, mantidos no repositório para facilitar a inspeção dos resultados sem exigir nova chamada à API.

## Arquivos

| Padrão | Descrição |
|---|---|
| `cnh_run_*.json` | Saídas estruturadas em JSON para a amostra de CNH |
| `invoice_run_*.json` | Saídas estruturadas em JSON para a amostra de fatura de energia |
| `paper_run_*.md` | Saídas em Markdown para o documento técnico extenso |
| `*_time.txt` | Tempo de execução medido em cada rodada |

## Privacidade

Saídas geradas a partir de documentos sintéticos ou públicos. Nenhum dado pessoal real, dado de cliente ou informação financeira real foi armazenado nesta pasta.
